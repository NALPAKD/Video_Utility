To filter by category in Access and export:
1. Create a query: SELECT Category, Title, iframe FROM YourTable WHERE Category = 'YourCategoryName'
2. Run the query and copy all results
3. Paste the results into the text area above
4. Or export query results to Excel, then copy from Excel

To get all categories: Use query SELECT DISTINCT Category FROM YourTable ORDER BY Category
